# Secure Expenses Tracker

A React Native (Expo) application focused on privacy-first personal expense tracking with a custom-built authentication system.

Built using **Expo + React Native + TypeScript** with a clean scalable architecture.

## Features (Planned & In Progress)

- Expo + TypeScript setup
- Clean project architecture (`src/navigation`, `src/screens`, `src/context`)
- Custom authentication system (no third-party auth providers)
- Secure token storage using Expo SecureStore
- User registration & login flow
- Expense management
- Categories & analytics
- OTP Pin using SMTP for 2FA

## Tech Stack

- **React Native**
- **Expo**
- **TypeScript**
- **React Navigation**
- **Expo SecureStore**

## A guide to start project

Clone the repository: 
```bash
git clone https://github.com/adiseshan1505/ExpensesTracker.git

cd SecureExpenseTracker
```

Install dependencies:

```bash
npm install
```

Start Expo: 
```bash
npx expo start --tunnel
```

Open using **Expo Go** on your mobile device.
